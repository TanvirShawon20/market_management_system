Step 8: Create User

user1.budget

1000

Step 9: Create Item
item1 = Item(
    name='IPhone 10',
    price=500,
    barcode='123654789123',
    description='DESC1'
)
Step 10: Assign Owner
item1.owned_user = user1
Step 11: Add Objects
db.session.add(user1)
db.session.add(item1)
Step 12: Commit
db.session.commit()

No error = success.

Step 13: Verify User
User.query.all()

You should see something like:

[<User 1>]
Step 14: Verify Item
Item.query.all()

You should see:

[Item IPhone 10]
Step 15: Verify Relationship

Check owner id:

item1.owner

Expected:

1

Check owner object:

item1.owned_user

Expected:

<User 1>

Check user's items:

user1.items

Expected:

[Item IPhone 10]
